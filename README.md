# To-Do List App

A vanilla JavaScript task manager built during my 180-day backend development sprint. No frameworks, no tutorials, no AI assistance. Hand-coded DOM manipulation and localStorage persistence.

## 🚀 Live Demo
[View on Netlify](https://newiup-todolist.netlify.app/)

## 🛠 Tech Stack
- HTML5
- CSS3 (Flexbox, Responsive Design)
- Vanilla JavaScript (DOM Manipulation)
- localStorage API
- Deployed on Netlify

## ✨ Features
- Add tasks via button click or Enter key
- Delete individual tasks
- Tasks persist after page refresh (localStorage)
- Live task counter
- Empty input validation
- Clean, minimal design

## 📚 What I Learned
- DOM manipulation (`createElement`, `appendChild`, `textContent`)
- Event listeners (`click`, `keypress`)
- Array methods (`push`, `splice`, `forEach`)
- Browser storage (`localStorage.setItem`, `getItem`, `JSON.stringify`, `JSON.parse`)
- Rendering data to screen dynamically
